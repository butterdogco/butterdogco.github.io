# butterdogco.github.io

This is the official repository for the ButterDogCo website, located at <https://butterdogco.com/>. The website is a static website made with standard web technologies and built with [gulp.js](https://gulpjs.com/), which minifies our HTML and inserts certain elements, such as the footer, into every page to prevent client-side creation.

## What's ButterDogCo?

ButterDogCo was founded by Keegan in the 7th grade. The website began with a simple search engine on Google Sites, originally called "ButterDog Search." The site has since expanded into different areas and has eventually become what it is now. We no longer rely on Google Sites, and every new website is entirely custom-made.
